# SDG4People

## Sustainable Development Goals

## Getting Set Up Locally

`bundle install`

`npm install`

`rake db:create`

`rake db:migrate`

### Development

To run locally: `./start.sh`

Navigate to `http://localhost:3000`.

### Testing

To run entire test suite: `./test.sh`

To run RSpec tests: `rspec`

To run Enzyme (React) tests: `npm run test`
